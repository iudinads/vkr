## Деплой на VPS

1. Купить домен и арендовать VPS.
2. Открыть файл `deploy.sh` и заполнить переменные, помеченные комментариями `# VPS_ЗАМЕНИТЕ_МЕНЯ: ...`.
3. Запустить деплой:
   ```bash
   chmod +x deploy.sh
   ./deploy.sh
   ```
4. Проверить, что приложение доступно по адресу `https://ваш-домен`.

Дополнительно после деплоя можно проверить API тестами:

```bash
DEPLOY_BASE_URL=https://ваш-домен pytest -q test_deploy.py
```
